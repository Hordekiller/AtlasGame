# اطلس | AtlasGame 🏛️

> یک بازی استراتژیک شهرسازی و امپراتوری‌سازی الهام گرفته از Ikariam — ساخته شده با **Godot 4.7**

---

## 🌍 نمای کلی

**اطلس** یک بازی استراتژیک چندنفره/تک‌نفره است که شما را به مقام فرمانروایی یک امپراتوری باستانی می‌رساند. جزایر را کشف کنید، شهر بسازید، اقتصاد را مدیریت کنید، فناوری تحقیق کنید و با دشمنان بجنگید.

---

## ✨ ویژگی‌ها

### 🏙️ شهرسازی
- **شهرهای متعدد** — استعمار جزایر مختلف و مدیریت همزمان چندین شهر
- **۳۰+ نوع ساختمان** — از معدن چوب و مزرعه گرفته تا کاخ و کارخانه ماشین پرنده
- **سیستم کارگر** — تخصیص کارگر به هر ساختمان برای بهینه‌سازی تولید
- **گرید شهری** — چیدمان آزاد ساختمان‌ها در شهر

### 📈 اقتصاد
- **۹ منبع اصلی** — چوب، مرمر، شیشه، شراب، طلا، غذا، سنگ، کریستال، گوگرد
- **سیستم رضایت (Happiness)** — افزایش جمعیت با شراب، میخانه، موزه و معبد
- **سیستم فساد (Corruption)** — هر مستعمره جدید فساد می‌آورد، فرمانداری آن را کاهش می‌دهد
- **انبار و ظرفیت** — ذخیره‌سازی منابع با انبار و زباله‌دان
- **تخفیف ساختمانی** — نجار، معمار، عینک‌ساز، آتشبازی و شراب‌فشاری

### 🔬 درخت تحقیق
- **۴ شاخه اصلی** — اقتصاد، نظامی، علم، دریانوردی
- **۳۵+ فناوری** — از چوب‌بری پیشرفته تا موشک‌شناسی و ناو هواپیمابر
- **پیش‌نیاز و لایه‌بندی** — تحقیقات لایه‌ای با پیش‌نیازهای هوشمند
- **تولید امتیاز پژوهش** — با ساختن آکادمی و کارگاه کریستال

### ⚔️ نبرد
- **سیستم نبرد نوبتی** — ۴ فاز: دریایی → بمباران → زمینی → محاصره
- **۳ صف جنگی** — خط مقدم، خط عقب، توپخانه
- **دیوار دفاعی** — دیوار شهر با قابلیت ارتقا و تخریب
- **واحدهای متنوع** — تیرانداز، هوپلیت، غول بخاری، منجنیق، خمپاره، آشپز، پزشک
- **روحیه سربازان** — آشپز روحیه می‌دهد، پزشک زخمی‌ها را مداوا می‌کند

### 🚢 نیروی دریایی
- **۱۲ نوع کشتی جنگی** — بالیستا، منجنیق، قوچ‌دار، آتشین، پارویی، غواص، موشکی، بخاری زره‌پوش
- **ناو هواپیمابر** — حمل واحدهای هوایی (ژیروکوپتر و بالن)
- **کشتی تدارکاتی** — حمل نیروی زمینی در دریا

### 🕵️ جاسوسی
- **۶ نوع مأموریت** — خرابکاری، سرقت منابع، سرقت تحقیق، شناسایی، تحریک شورش، مسموم کردن آب
- **مخفیگاه** — هر سطح مأموریت‌های همزمان بیشتری فعال می‌کند
- **ریسک کشف و مرگ** — جاسوس ممکن است کشف و اعدام شود

### 🤝 تجارت
- **بازار** — خرید و فروش منابع بین بازیکنان
- **بازرگانان NPC** — بازدید دوره‌ای تاجران با پیشنهادات ویژه
- **مسیرهای تجاری** — ارسال منابع بین شهرهای خودی

### 📱 موبایل
- **طراحی کاملاً لنداسکیپ** — بهینه شده برای تبلت و موبایل افقی
- **تاچ پشتیبانی کامل** — پینچ زوم، درگ با انگشت، دکمه‌های بزرگ لمسی
- **HUD ریسپانسیو** — نوار منابع ۵۶px، نوار ساختمان ۸۸px

## 🎮 گیم‌پلی

### شروع بازی
1. تالار شهر خود را بسازید
2. منابع اولیه (چوب، غذا) را تأمین کنید
3. جمعیت را افزایش دهید
4. تحقیقات ابتدایی را شروع کنید

### رشد و توسعه
1. جزایر جدید را با **کاخ** مستعمره کنید
2. زنجیره تولید را کامل کنید (چوب → شیشه → کریستال → تحقیق)
3. رضایت را با **میخانه** و **موزه** بالا نگه دارید
4. **اقامتگاه فرماندار** برای کاهش فساد بسازید

### نظامی
1. **پادگان** برای سربازان زمینی
2. **بندر** و **کارخانه کشتی‌سازی** برای نیروی دریایی
3. **دیوار دفاعی** برای محافظت از شهر
4. **برج کیمیاگر** برای تولید گوگرد (مهمات)

## 🛠️ تکنولوژی

| بخش | توضیح |
|------|--------|
| **موتور** | Godot 4.7 stable |
| **زبان** | GDScript |
| **پلتفرم هدف** | Android (Landscape), Web, PC |
| **رزولوشن** | 1920×1080 (canvas_items + expand) |
| **Assets** | فرمت PNG/JPG در مسیر `Assets/Textures/` |
| **تم UI** | StyleBoxFlat اختصاصی با `UITheme.gd` |

## 📂 ساختار پروژه

```
GameMB/
├── Assets/
│   ├── Shaders/          # شیدرهای GLSL
│   └── Textures/         # تمام تصاویر بازی
│       ├── Buildings/    # اسپرایت ساختمان‌ها
│       ├── Resources/    # آیکون منابع
│       ├── Units/        # اسپرایت واحدها
│       ├── UI/           # المان‌های رابط کاربری
│       └── Environment/  # بک‌گراند و محیط
├── Globals/              # اسکریپت‌های سراسری
├── Scenes/
│   ├── Menu/             # منوی اصلی
│   ├── Game/             # صحنه‌های بازی
│   └── Building/         # کامپوننت ساختمان
└── Scripts/
    ├── Core/             # هسته بازی
    ├── Military/         # سیستم نظامی
    ├── Systems/          # سیستم‌های پیشرفته
    ├── UI/               # کامپوننت‌های UI
    ├── World/            # منطق جهان
    └── City/             # منطق شهر
```

```bash
git clone https://github.com/Hordekiller/AtlasGame.git

godot project.godot
```
